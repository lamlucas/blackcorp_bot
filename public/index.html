Lọc Chat ID

Bấm « Lấy Chat ID nhóm » để nạp danh sách. « Tên đại lý » trên form phải trùng tên tab Sheet — có thể sửa sau khi Thêm.

Gửi chi phí

Sheet chi phí: tab TINH_TIEN1 … TINH_TIEN6 (mỗi dòng từ dòng 2 một tin nhóm + QR; dừng khi cột A–I trống). Cột D = tên khách khớp cột A tab CONG_NO (sheet công nợ) để tính TỔNG THU. Bot ghi TỔNG THU vào cột J. Chat ID nhóm: ô B1 từng tab hoặc form « Đại lý » (tên đại lý = đúng tên tab, ví dụ TINH_TIEN1).

LINK FILE (theo tab)

LINK FILE 1 → tab TINH_TIEN1, … LINK FILE 6 → tab TINH_TIEN6. Mọi dòng trong tab đó dùng chung link đã nhập.

Thông báo công nợ

Dữ liệu công nợ đọc từ tab CONG_NO trên bảng THEO DÕI TÀI KHOẢN + TÍNH TIỀN (cột A = tên tab trên Sheet chi phí vd TINH_TIEN1, cột B = nợ). Bot gửi theo Chat ID (B1 tab hoặc form Đại lý). Cron 22:00 giờ Việt Nam.
